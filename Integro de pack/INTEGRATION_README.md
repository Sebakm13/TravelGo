
# TRAVELGO INTEGRATION PACK

Este ZIP contiene toda la integración lista del microservicio de destinos.

## Cómo instalarlo
1. Descarga este ZIP.
2. Descomprímelo.
3. Copia la carpeta **app/** de este ZIP dentro de tu proyecto Android.
4. Acepta "Replace files".
5. ¡Listo! La integración se aplica automáticamente.

## Qué incluye
- ApiService modificado
- DTOs
- Repository
- ViewModel
- Pantalla Compose
- Retrofit apuntando al microservicio

Microservicio debe correr en **http://10.0.2.2:8081/**
